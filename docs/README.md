## V2.0 核心变更速览

| 变更维度 | V1.0 | V2.0 |
|---|---|---|
| 架构模式 | 单店系统 | 多商户 SaaS（平台+商户两层） |
| 管理端 | 1 套（商户管理） | 2 套（平台管理端 + 商户管理端） |
| 数据隔离 | 无 | 四层隔离（DB/Cache/文件/API） |
| 点餐推荐 | RAG + LLM 简单推荐 | 混合推荐引擎（CF+CB+KG+LLM 策略调控） |
| 库存预测 | 基础消耗预测 | 多因子时序预测 + LLM 校准 + 供应商匹配 |
| 客服处理 | FAQ + 简单分流 | 深层意图 + 情绪趋势 + 流失预警 + 挽回话术 |
| 经营分析 | KPI 汇总 + LLM 洞察 | 异常检测 + 归因分析 + What-if 模拟 + 菜单工程 + RFM |
| 小程序 | 7 页 | 9 页（新增选店首页 + 商户搜索） |
| 开发周期 | — | 2 周冲刺（3 人团队，14 天详细计划） |
